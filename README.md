## 5. 사용 모델 & 모델 선정
* MediaPipe 'Blazepose model', OpenCV 'Canny model'
* PCK @ 0.2 로 성능 비교 결과 '춤'에 관한 Task들은 'Blazepose' 모델이 성능 최고
* OpenPose vs BlazePose Fps 비교 : BlazePose가 25~75배 이상 높은 것으로 확인
